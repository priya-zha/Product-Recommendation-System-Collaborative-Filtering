# 🛍️ Product Recommendation System — Collaborative Filtering

A machine learning-based product recommendation system that suggests products to users based on the purchase history of similar users. Uses collaborative filtering techniques to generate personalized recommendations.

---

## 📌 Description

This project implements a **collaborative filtering** recommendation engine — it analyzes what other users with similar purchase behavior have bought, and recommends those products to the current user. This is the same underlying technique used by major e-commerce platforms like Amazon and Netflix to drive personalized recommendations.

## ⚙️ Setup Instructions

### 1. Clone the Repository
```bash
git clone https://github.com/priya-zha/product_recommendation_system.git
cd product_recommendation_system
```

### 2. Install Dependencies
```bash
pip install pandas numpy scikit-learn jupyter
```

### 3. Launch the Notebook
```bash
jupyter notebook "product_recommend_based_on_other_user's_purchase.ipynb"
```

---

## 🚀 Usage

1. Open the notebook in Jupyter or Google Colab
2. Run cells in order to:
   - Load and preprocess user purchase data
   - Compute user similarity scores
   - Generate product recommendations for a given user based on similar users' purchases
   - Evaluate recommendation quality
